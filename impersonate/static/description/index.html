Impersonate

Allows to impersonate other users.

This module will provide quick and passwordless switching between allowed users.
You can use it in development to try other users with different accesses or in production to supervise users and see everything as they see.

Installation

Simply locate the module and click install. No other dependencies required.

Configuration

Impersonate anyone

To allow for user to impersonate anyone simply select "Impersonate Anyone".

Warning: User can impersonate even administrator/superuser account. Be absolutely sure before granting this access!

Impersonate selected users

To allow for user to impersonate selected users - simply add wanted users to "Impersonate Users" list located at "Preferences" tab.

Usage

To start impersonating: Select "Impersonate" from user menu.
Select user you wish to impersonate and click "Impersonate".

After successful impersonation you will see icon that will indicate impersonation process.

To stop impersonating click on "Impersonate" icon or select "Stop Impersonation" from user menu.
